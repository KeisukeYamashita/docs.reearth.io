# Re:Earth User Manual

欢迎来到reearth的user manual。这里是教大家如何使用reearth的地方。

目前reearth正处于紧张的开发阶段，很多新功能会加入其中，也有很多旧功能会不断更新，还有很多想法正在孕育中，因此这个手册也许不是一个完美的方案，但是我们还是想在教会大家使用的同时，分享我们对于项目的思考，并在开发中不断探索方向。
