当你完成登陆之后，首先看到的便是[Dashboard]。点击 `New project` 按钮，将会出现初始化项目的对话框。

<Image path={require('./img/2-1.png').default} />

<Image path={require('./img/2-2.png').default} />

<Image path={require('./img/2-3.png').default} />

<Order>
  <ListByNumber>
    点击 New project 按钮，打开初始化项目的对话框
  </ListByNumber>
  <ListByNumber>
    填写项目名称
  </ListByNumber>
  <ListByNumber>
    填写项目简介
  </ListByNumber>
  <ListByNumber>
    选择并添加Thumbnail image
  </ListByNumber>
  <ListByNumber>
    确认创建新项目
  </ListByNumber>
  <ListByNumber>
    完成之后，你会发现一个新的项目的卡片出现在项目列表中
  </ListByNumber>
</Order>

至此你已经创建了你的第一个Re:Earth项目。点击项目卡片的编辑按钮，我们便可以进入我们刚刚创建的项目了。
