因为Re:Earth还处于内部开发的阶段，因为我们还未支持公开的注册账号。所以请通过【这里】联系我们，我们将会为你开通一个新账号。

获得账号并完成验证之后，请打开[https://app.reearth.io/](https://app.reearth.io/)，输入你的账号信息。

<Image path={require('./img/1-1.png').default} />

正常情况下，由此你便可以开始使用Re：Earth了。如果出现了各种错误或者无法登陆的情况，请通过「这里」联系我们，我们会帮助你排除问题。
