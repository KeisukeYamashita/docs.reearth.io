sidebar_position: 6

当你在地图上放置了marker之后，你便想呈现更多关于这个marker的信息，[infobox]正好是你所需要的工具。

## 创建infobox

[此处应该有牛逼的步骤图]

1.选中layer
2.选择infobox标签
3.点击 create infobox 按钮创建infobox

## 添加block

现在我们有一个空的infobox，想要在其中添加信息时，我们需要创建[block]。

[此处应该有牛逼的步骤图]

1.移动鼠标到创建按钮处
2.选择想使用的block
3.选中block
4.修改block参数，为其添加内容

通过infobox我们便可以为layer添加各种信息。你可以尝试一下各种不同的block，看看能够用他们来做些什么。
