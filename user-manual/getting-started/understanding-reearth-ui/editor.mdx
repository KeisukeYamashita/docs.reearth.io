Editor是实际上编辑项目的页面，大部分的时间你将在这个页面工作。

## 功能区域

[帅气照片，显示出各个功能区域]

Editor主要分为五大功能区域：
- Header 起到导航和发布项目的作用，详情请查看[Header]
- [Left Panel] 是管理项目资源和数据的地方
- [Tool Bar] 给地图添加元素的工具区域
- [Right Panel] 则是调整参数的区域
- [Viewer] 则是项目显示的区域

基本上的设计想法是，你可以在使用Tool Bar中的工具来创建Layer，然后在Left Panel中管理他们，并且在Right Panel中调节参数，在Viewer中查看效果。当然不同的功能模块还有各自具体的操作方式。

## Left Panel

在Re:Earth中有各种不同类型的资源需要管理。比如在地图上呈现的Layer，在界面中呈现的Widget等等，他们都可以在Left Panel中查看。Left Panel有两个主要的标签 Outline 和 Dataset。

### Outline

Outline 用来呈现和管理项目的资源。

[帅气的图]

A.Scene对象，用来调节一些场景和项目的参数，具体请查看[Scene]
B.Widget列表，用来管理项目Widget，具体请查看[Wigets & Align system]
C.Layer列表，用来管理项目的Layer，具体请查看[Layers]
D.Cluster列表，具体请查看[Cluster]

### Dataset

[帅气且自卑的Dataset照片]

切换至Dataset标签，你可以查看和管理当前项目所引用的dataset。

[瘦弱的Tool Bar照片]
