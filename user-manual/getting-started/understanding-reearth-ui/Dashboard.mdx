Dashboard是你登陆Re:Earth之后，第一看到的页面。这个页面主要提供了一些快速的功能，帮助你迅速开始你的工作。

## 功能区域介绍

[帅气牛逼的图]

A.workspace Secion 是关于Workspace的快速操作区域
B.当前所在workspace
C.成员列表
D.设置按钮，带你前往Workspace Setting页面
E.快速开始区域
F.带你前往用户手册
G.New project 按钮将会开始创建新项目
H.New workspace 按钮将会开始创建新workspace
I.项目一览区域
J.[项目卡片]

## 项目卡片

当前workspace下的项目（不包括已经archived的项目）都会以项目卡片的形式出现在列表之中。光标移动到卡片上边，更多的信息将会显示。

[帅气的两张对比图]

A.项目名称
B.发布情况会显示当前项目是否已经公开
C.Edit按钮，将会带你进入Editor，开始编辑你的项目
D.进入项目的Preview的按钮
E.进入项目的setting 页面的按钮
F.项目的介绍

## 将来的版本

在将来的版本中，我们还会在Dashboard页面中添加一些新的功能，比如开发log，新功能介绍，社区新闻等等。
