Header主要负责导航，同时也可以快速地完成一些功能，比如发布项目等等。
在RE:Earth的每个页面，都会有Header的存在。但是在不同的页面，Header可使用的功能稍有不同。这其中属Editor页面的Header功能最为齐全，因为我们以它为例介绍。

[此处应该有帅气的header照片]

A.前往Dashboard的按钮
B.前往Editor的按钮
C.用户头像（功能还在开发中...）
D.用户名
E.当前所处的workspace
F.[账户以及workspace功能下拉列表]
G.当前正在编辑的项目名
H.[关于项目的下拉列表]
I.项目[预览]按钮
J.项目[发布]功能按钮

## 账户以及workspace功能下拉列表

[此处应该有点开图]

A.Account setting 带你前往Account setting SettingPage
B.Switch worspace 帮你在不同workspace中切换
C.当前所处的workspace
D.Manage workspace 带你前往workspace一览的页面
E.New workspace 让你可以在任何时候新建一个workspace
F.登出你当前的账户
